# Government Contract Opportunity Scanner and Proposal Tracker Discord Bot

A Discord bot for federal contractors, small business owners, and GovCon consultants that monitors SAM.gov and USASpending.gov for new contract opportunities matching your NAICS codes and capability statements. Sends real-time alerts for new solicitations, tracks proposal due dates, logs teaming partner outreach, and posts daily GovCon opportunity digests to your team channel.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
